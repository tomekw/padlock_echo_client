# Padlock Echo Client implementation

[Padlock](https://github.com/tomekw/padlock) is LibreSSL's [libtls](https://man.openbsd.org/tls_init.3) in Ada.

## Status

This is alpha software. I'm actively working it. YMMV.

Tested on Linux x86_64, MacOS ARM and Windows x86_64.

## Usage

Install [tada](https://github.com/tomekw/tada).

See [example echo server implementation](https://github.com/tomekw/padlock_echo_server)

Run both with:

``` shell
tada run
```
